Step 2 of 7 — Detect the stars (centroids)

A blob detector finds every bright spot above the local background. Each blob's position is then refined to sub-pixel precision with a flux-weighted moment centroid — the intensity-weighted mean pixel:

x_c = ( Σ Iᵢ·xᵢ ) / ( Σ Iᵢ ) y_c = ( Σ Iᵢ·yᵢ ) / ( Σ Iᵢ ) Iᵢ = pixel intensity above the local background, summed over the blob. The centroid is a weighted mean, so it lands between pixels — far more precise than the nearest whole pixel.
