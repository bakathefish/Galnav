Step 3 of 7 — Identify the stars

The plate solution gives every pixel a sky coordinate, so each detected blob can be cross-matched to the Gaia catalog by position. We learn what each dot is and how far away it lies. Only the nearby stars — whose parallax shift is large enough to reveal the camera's position — are position-capable; the rest are known but too far to navigate by.
